# -# 算法学习与授课资料 (Algorithm Learning & Teaching Materials)

本仓库用于整理和归档算法相关的学习例题与授课资料，主要分为以下两个部分：

## 📁 目录结构

### 1. [算法例题](./算法例题)
这里包含在算法学习、Codeforces (CF)、AtCoder (ATC) 等竞技编程平台上的经典例题代码、模板和题解。
- 总结了常见的数据结构与算法模板。
- 记录了各类题型的解题思路与技巧。

### 2. [算法授课](./算法授课)
这里包含算法教学与授课的相关资料（如课件、讲义、代码示例等）。
- 适合初学者入门与进阶的代码讲解。
- 授课过程中的随堂练习与解析。

## 🚀 关于竞技编程模板的建议
在算法竞赛中，为了提高解题速度并减少低级 Bug，建议在本地编辑器中配置好缺省源（如 C++ 的快读快写、常用宏定义等），并将本仓库中的经典算法（如线段树、最短路、并查集等）配置为代码片段（Snippets）。
